    Replace the outer `Column(modifier = GlanceModifier.fillMaxSize()...)` block with a `LazyColumn` structured as follows:

    ```kotlin
    LazyColumn(modifier = GlanceModifier.fillMaxSize().background(ColorProvider(ZenSurface)).padding(12.dp)) {
        // Header row: app name + completion count
        item {
            Row(
                modifier = GlanceModifier.fillMaxWidth().padding(bottom = 4.dp),
                verticalAlignment = Alignment.CenterVertically
            ) {
                Text(
                    text = "ZenStack",
                    style = TextStyle(fontSize = 16.sp, fontWeight = FontWeight.Bold, color = ColorProvider(ZenIndigo))
                )
                Spacer(modifier = GlanceModifier.defaultWeight())
                Text(
                    text = "$completedCount/${priorityTasks.size}",
                    style = TextStyle(fontSize = 14.sp, fontWeight = FontWeight.Medium, color = ColorProvider(ZenGrayDark))
                )
            }
        }
        // Progress bar
        item {
            LinearProgressIndicator(
                progress = progress,
                modifier = GlanceModifier.fillMaxWidth().height(8.dp).padding(bottom = 12.dp),
                color = ColorProvider(ZenIndigo),
                backgroundColor = ColorProvider(Color.LightGray.copy(alpha = 0.2f))
            )
        }
        // Priority section header
        item {
            Text(
                text = "Priority",
                style = TextStyle(fontWeight = FontWeight.Bold, fontSize = 13.sp, color = ColorProvider(ZenGrayDark)),
                modifier = GlanceModifier.padding(bottom = 4.dp)
            )
        }
        // Priority task rows â€” no cap, all tasks
        items(priorityTasks) { task ->
            TaskItem(task)
        }
        // Brain dump section header (only shown when non-priority tasks exist)
        if (otherTasks.isNotEmpty()) {
            item {
                Text(
                    text = "Brain Dump",
                    style = TextStyle(fontSize = 11.sp, color = ColorProvider(ZenGrayDark.copy(alpha = 0.6f))),
                    modifier = GlanceModifier.padding(top = 12.dp, bottom = 4.dp)
                )
            }
            // Brain dump task rows â€” no cap, all tasks
            items(otherTasks) { task ->
                TaskItem(task, isSmall = true)
            }
        }
    }
    ```

    Remove the old outer `Column` block entirely. The `clickable(actionStartActivity(clickIntent))` modifier that was on the outer Column should be removed â€” individual `TaskItem` rows already handle clicks, and applying a widget-wide click conflicts with LazyColumn scrolling. The `clickIntent` variable and its construction can be removed too if nothing else references it.

    Keep `TaskItem` composable unchanged. Keep `ZenStackWidgetReceiver` unchanged.

    Note: `cornerRadius` cannot be applied to `LazyColumn` in Glance. Remove it (it was on the old Column). The widget host clips corners via the widget XML configuration instead.
