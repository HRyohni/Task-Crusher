    Add an enum above `ZenStackApp`:

    Add the `MainShell` composable after `ZenStackApp`:
    ```kotlin
    @Composable
    fun MainShell(
        tasks: List<Task>,
        onRestart: () -> Unit,
        onToggleComplete: (Task) -> Unit
    ) {
        var selectedTab by remember { mutableStateOf(MainTab.TODAY) }

        Scaffold(
            bottomBar = {
                NavigationBar {
                    NavigationBarItem(
                        selected = selectedTab == MainTab.TODAY,
                        onClick = { selectedTab = MainTab.TODAY },
                        icon = { Icon(Icons.Default.Today, contentDescription = "Today") },
                        label = { Text("Today") }
                    )
                    NavigationBarItem(
                        selected = selectedTab == MainTab.CALENDAR,
                        onClick = { selectedTab = MainTab.CALENDAR },
                        icon = { Icon(Icons.Default.CalendarMonth, contentDescription = "Calendar") },
                        label = { Text("Calendar") }
                    )
                    NavigationBarItem(
                        selected = selectedTab == MainTab.ACHIEVEMENTS,
                        onClick = { selectedTab = MainTab.ACHIEVEMENTS },
                        icon = { Icon(Icons.Default.EmojiEvents, contentDescription = "Achievements") },
                        label = { Text("Achievements") }
                    )
                }
            }
        ) { innerPadding ->
            Box(modifier = Modifier.padding(innerPadding).fillMaxSize()) {
                when (selectedTab) {
                    MainTab.TODAY -> CurrentTasksView(
                        tasks = tasks,
                        onRestart = onRestart,
                        onToggleComplete = onToggleComplete
                    )
                    MainTab.CALENDAR -> CalendarPlaceholder()
                    MainTab.ACHIEVEMENTS -> AchievementsPlaceholder()
                }
            }
        }
    }
    ```

    Add two placeholder composables after `MainShell`:
    ```kotlin
    @Composable
    fun CalendarPlaceholder() {
        Box(modifier = Modifier.fillMaxSize(), contentAlignment = Alignment.Center) {
            Text("Calendar — coming soon", style = MaterialTheme.typography.bodyLarge)
        }
    }

    @Composable
    fun AchievementsPlaceholder() {
        Box(modifier = Modifier.fillMaxSize(), contentAlignment = Alignment.Center) {
            Text("Achievements — coming soon", style = MaterialTheme.typography.bodyLarge)
        }
    }
    ```

    Update `ZenStackApp` else-branch:
    ```kotlin

        MainShell(
            tasks = savedTasks!!,
            onRestart = { showSetup = true },

        )
    }
    ```

    IMPORTANT: `CurrentTasksView` already has its own `Scaffold` with a FAB. Since `MainShell` provides an outer Scaffold for the NavigationBar, the inner Scaffold in `CurrentTasksView` will still render correctly — the inner Scaffold will nest inside the outer padding provided by `MainShell`'s Scaffold. Do NOT remove `CurrentTasksView`'s internal Scaffold; it handles the FAB independently.

    Remove `Today` icon import if it conflicts — use `Icons.Default.Today` (from material-icons-extended, already a dependency). If `Icons.Default.Today` does not resolve, substitute with `Icons.Default.Home`.
