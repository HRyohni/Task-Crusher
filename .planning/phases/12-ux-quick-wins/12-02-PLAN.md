**Change B — Filter completed tasks to today-only in CurrentTasksView:**

At the top of `CurrentTasksView`, after the existing `priorityTasks`/`otherTasks` splits, add a today-date filter for completed tasks. The `tasks` list passed in already contains all tasks. Add:

```kotlin
val today = java.time.LocalDate.now()
val zoneId = java.time.ZoneId.systemDefault()

fun Task.completedToday(): Boolean {
    val at = completedAt ?: return false
    return java.time.Instant.ofEpochMilli(at).atZone(zoneId).toLocalDate() == today
}

val priorityTasks = tasks.filter { it.isPriority && (it.status != "COMPLETED" || it.completedToday()) }
val otherTasks = tasks.filter { !it.isPriority && (it.status != "COMPLETED" || it.completedToday()) }
```

This replaces the existing `priorityTasks` and `otherTasks` val declarations (lines ~287-288). Note: `java.time.LocalDate` is already imported at the top of ZenStackApp.kt; `java.time.ZoneId` and `java.time.Instant` need to be added to imports.
