In `BrainDumpStep`, update the `OutlinedTextField` to support Enter/Done key submission:

1. Add two imports at the top of the file (after the existing imports block):
   ```
   import androidx.compose.ui.text.input.ImeAction
   import androidx.compose.foundation.text.KeyboardActions
   import androidx.compose.foundation.text.KeyboardOptions
   ```

2. Extract the submit logic into a local lambda inside `BrainDumpStep` (before the `Column`):
   ```kotlin
   val submitTask: () -> Unit = {
       if (text.isNotBlank()) {
           haptic.performHapticFeedback(HapticFeedbackType.TextHandleMove)
           onAddTask(text)
           text = ""
       }
   }
   ```

3. Add `keyboardOptions` and `keyboardActions` parameters to the existing `OutlinedTextField` in `BrainDumpStep`:
   ```kotlin
   keyboardOptions = KeyboardOptions(imeAction = ImeAction.Done),
   keyboardActions = KeyboardActions(onDone = { submitTask() }),
   ```

4. Update the existing `trailingIcon` `IconButton` onClick to call `submitTask()` instead of the inlined logic (removes duplication).

The TextField already has `modifier`, `value`, `onValueChange`, `textStyle`, `placeholder`, `colors`, and `trailingIcon` — insert `keyboardOptions` and `keyboardActions` as additional named parameters.
